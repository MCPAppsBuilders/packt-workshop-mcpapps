# Module 02 - Architecture

Basic MCP server with an interactive view, demonstrating the core MCP primitives: **tools**, **resources**, and **prompts**.

## Structure

```
02-architecture/
├── mcp-server/      # Python MCP server (FastMCP + uv)
│   ├── server.py    # 2 tools (add, subtract), 2 resources, 1 prompt
│   ├── resources/   # Static resource files (info.txt, config.json)
│   └── assets/      # Built view HTML (generated by the view build)
└── view/            # TypeScript React UI (Vite + MCP Apps SDK)
    └── src/
        └── mcp-app.tsx  # Displays tool input (A, B) and result (A+B)
```

## Build

### 1. Build the view

```bash
cd view
npm install
npm run build
```

This generates `mcp-server/assets/basic_add.html` (a single-file HTML with all JS/CSS inlined).

### 2. Run the MCP server

```bash
cd mcp-server
uv run python server.py
```

### Development

To work on the view with hot-reload:

```bash
cd view
npm run dev
```
